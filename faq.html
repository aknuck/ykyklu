FAQ

What is Hawk Yak

It is a browser version of the Lehigh Yak feed, viewable only when connected to the Lehigh wifi.

What can it do?

It is currently view only, but we are planning on adding the ability to post yaks and comments. We are also planning to allow you filter all yaks with certain hashtags, as well as add images to yaks.

Who made this?

This was created by roommates Adam Knuckey and Deep Sheth in November of 2014. The frontend was written by Deep in HTML, CSS, and Javascript. The backend was written by Adam in Python using the web.py web framework. The design was a collaboration from both of us.
